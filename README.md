# 1942

Welcome to the game 1942, a recreation of the 1984 vertically scrolling shooter game by Capcom.

Itch: https://jebbbo.itch.io/1942

**_In this game the goal is to defeat all the enemy planes and make it to the end of each level. Make sure to keep shooting down those planes as the waves don't stop!_**

## How To Play

You can use WASD to move and SPACE to shoot!
There are two levels - good luck!

## Installation

You can download the latest version of the game from the releases section.

The game is also avaiable to download from [it's itch page](https://jebbbo.itch.io/1942)
